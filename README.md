# Gestao de Futebol - Algoritmos

Plataforma para gestão e análise estatística de futebol. Auxilia comissões técnicas na avaliação de atletas e métricas de jogos.
• Público: Treinadores e analistas de scout.
• Funcionalidades: Cálculo de média de gols, aproveitamento de times e classificação por idade/desempenho.
• Diferenciais: Lógica automatizada e critérios precisos de triagem.
